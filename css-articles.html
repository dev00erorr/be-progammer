اساسيات CSS

مقدمة في CSS

CSS (Cascading Style Sheets) هي لغة تصميم تُستخدم لتنسيق مظهر صفحات الويب. تعمل CSS جنبًا إلى جنب مع HTML لتعطي صفحات الويب شكلها وجماليتها من حيث الألوان، الخطوط، التخطيطات، وحجم العناصر.

أهم خصائص CSS

  • color: لتغيير لون النص.
  • font-size: لتحديد حجم النص.
  • background-color: لتحديد لون الخلفية.
  • margin: لتحديد المسافة خارج العنصر.
  • padding: لتحديد المسافة داخل العنصر.
  • display: لتحديد كيفية عرض العناصر.
